# Padrões de projeto com TypeScript

Este é o repositório da seção de Padrões de Projeto disponível gratuitamente em:

https://www.youtube.com/watch?v=MqddY6Ochkc&list=PLbIBj8vQhvm0VY5YrMrafWaQY2EnJ3j8H
